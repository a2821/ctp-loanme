Steps to work with Github

Step 1: Clone the repository in your local machine using  https://github.com/ranbala000/ctp-loanme.git
Step 2: Switch to branch ‘pre-final’ using git checkout pre-final
Step 3: Create a new branch to work on your tasks independently using git –b checkout <branch-name>
Step 4: After completing your work on your local machine, push it
